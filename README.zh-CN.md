一、基础行为规范

1. 严禁使用作弊客户端、外挂、修改器，包括但不限于飞行、透视、物品刷取、秒杀等作弊行为；禁止利用模组漏洞（如女仆任务刷物、存储模组复制资源）获取不正当利益，发现漏洞需及时向管理员反馈。
2. 禁止发布广告、涉政、色情、暴力等不良信息，不得进行人身攻击、辱骂、造谣等恶意言论；服务器内禁止刷屏、恶意@全体玩家，保持聊天频道整洁。
3. 尊重其他开拓者的劳动成果，禁止恶意破坏他人建筑、农田、红石机器、女仆机器人、存储系统等设施；未经允许不得随意移动他人放置的模组道具（如女仆核心、存储磁盘、魔法符文）。
4. 禁止搭建高频红石、卡顿机器（如无限循环的女仆任务机、超大规模存储集群），若装置导致服务器TPS大幅下降，管理员有权要求拆除并限制相关模组功能使用。

二、模组专属玩法规则

（一）女仆模组（Touhou Little Maid）相关

1. 合理设置女仆任务，禁止利用 maid_useful_task / maid_storage_manager 模组的任务机制刷取物品、经验，或让女仆执行无限重复的卡顿任务。
2. 禁止恶意攻击、杀死其他开拓者的女仆机器人，若需借用他人女仆，必须提前征得主人同意；女仆的装备、核心配件禁止盗取。
3. 女仆的生成与管理需遵守服务器实体数量限制，单个区块内女仆数量不得超过10个，避免实体过多造成服务器卡顿。

（二）农业/美食模组（FarmersDelight、Aquaculture等）相关

1. 合理开垦农田、养殖水产，禁止在公共区域（如出生点、维度入口）无节制搭建超大规模农场/渔场，占用公共资源。
2. 食材、料理交易需秉持诚信原则，禁止使用 CraftTweaker 模组修改配方后欺诈其他开拓者，或利用 diet 模组的饮食机制恶意干扰他人游戏体验。
3. 禁止在水源区域投放模组内污染道具，破坏水产养殖与海洋生态（OceansDelight）环境。

（三）魔法模组（Ars Nouveau）相关

1. 魔法符文、法术构建需合规使用，禁止释放大规模破坏性魔法（如区域秒杀、地形摧毁）攻击其他开拓者或破坏公共地形。
2. 魔法道具（如法术书、魔晶）的交易与赠予需自愿，禁止抢夺、骗取他人的稀有魔法物品；禁止利用魔法漏洞无限生成魔能、符文。

（四）存储/工业模组（Refined Storage、IndustrialUpgrade）相关

1. 使用精致存储模组时，禁止搭建跨维度的无限资源复制装置，或利用存储节点漏洞刷取磁盘、物资； ExtraDisks 扩展磁盘需合规使用，禁止滥用存储权限。
2. 工业模组的发电机、反应堆需合理规划摆放位置，禁止在开拓者基地附近搭建未防护的核反应堆等高风险装置，避免爆炸破坏地形与建筑。

（五）维度探索模组（Twilight Forest、The Undergarden）相关

1. 暮色森林、地下花园的BOSS击杀、稀有资源采集需公平竞争，禁止使用模组道具恶意卡位、秒杀BOSS，或垄断钻石矿脉、暮色森林城堡等关键资源点。
2. 禁止在维度入口处搭建阻碍通行的建筑，或利用维度传送机制恶意传送其他开拓者至地下花园深渊、暮色森林巫妖塔等危险区域。

三、领地与交互规则

1. 服务器已启用领地保护模组，开拓者可通过模组指令划定专属领地，也可放置告示牌标注建筑、模组设施所属权；若发现他人恶意侵占、破坏领地内的女仆、存储、农场等设施，可向管理员提交申诉，核实后将对违规者进行处罚。
2. 与其他开拓者进行模组道具、资源交易时，需秉持诚信原则，禁止欺诈、哄抬稀有魔法符文、女仆核心等道具物价，或使用虚假物品交易；若产生交易纠纷，可请求管理员介入调解，管理员将根据实际情况判定。
3. 禁止未经主人允许擅自进入他人未公开的建筑、基地、领地，或查看他人的存储系统、女仆任务配置；若需进入/查看，需提前通过游戏内聊天、服务器社群征得同意，违规闯入视为恶意骚扰行为。
4. 组队探索维度、挑战BOSS时，需遵守团队约定分配战利品，禁止私吞暮色森林骑士剑、Ars Nouveau史诗法术书等稀有模组道具。

四、处罚机制

1. 首次违规：管理员口头警告，责令整改违规行为（如拆除卡顿机器、归还盗取的模组道具、恢复被破坏的建筑）。
2. 二次违规：临时封禁账号1-7天，清除违规所得的模组物品、资源；若涉及恶意破坏，需赔偿受害者相应损失。
3. 严重违规：多次作弊、利用模组漏洞破坏服务器、恶意攻击其他开拓者等行为，将被永久封禁账号、清除相关游戏数据，且不接受申诉。
4. 特殊情形：若因模组BUG导致无心违规，主动向管理员说明情况并配合修复的，可减轻或免除处罚。

1. 服务器管理员拥有规则解释权与违规判定权，若对处罚有异议，可向管理员提交书面申诉（附游戏内截图、视频等证据）。
2. 服务器会定期备份数据，但不承担开拓者因自身操作（如误删女仆、格式化存储磁盘）造成的损失，建议自行备份重要模组道具。
3. 欢迎开拓者向管理员提出模组玩法优化、服务器功能改进的合理建议，共同完善本服的游戏环境。
